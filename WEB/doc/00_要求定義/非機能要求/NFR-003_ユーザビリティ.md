# NFR-003: ユーザビリティ

## 概要

WEB版として使いやすいUI/UXを提供する。

## 要求一覧

### NFR-003-01: レスポンシブデザイン

- デスクトップ・タブレット・モバイルに対応
- 最小対応幅: 375px（スマートフォン）

### NFR-003-02: ブラウザ互換性

- Chrome, Firefox, Safari, Edge の最新2バージョン

### NFR-003-03: アクセシビリティ

- WAI-ARIA準拠
- キーボードナビゲーション対応
- スクリーンリーダー対応

### NFR-003-04: ローディング状態

- データ読み込み・計算中のローディングインジケータ表示
- エラー発生時のユーザーフレンドリーなメッセージ

### NFR-003-05: 操作性

- 元のWPFアプリのタブ切替・サイドバー操作に相当するナビゲーション
- フィルタ操作の直感的なUI（トグル、ドロップダウン、日付ピッカー）

## テスト観点

- 各ブレークポイントでのレイアウト崩れなし
- 主要ブラウザでの表示・動作確認
- キーボードのみでの全機能操作
- ローディング→データ表示の遷移
