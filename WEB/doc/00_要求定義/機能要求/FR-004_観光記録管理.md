# FR-004: 観光記録管理

## 概要

旅行中の観光活動に関する記録を管理する。

## 要求一覧

### FR-004-01: 観光記録の登録

- 以下の項目で観光記録を登録できる
  - コンテキスト（活動名/説明、必須）
  - 観光タイプ（SightseeigType列挙値）
  - 日付（必須）
  - 国（必須）、地域（任意）
  - 料金（必須）、通貨（必須）
  - メモ（任意）

### FR-004-02: キーワードベースの自動タイプ分類

- コンテキスト文字列に含まれるキーワードで自動分類:
  - "BEACH" → Beach
  - "MUSEUM" → Museum
  - "CHURCH" → Church
  - "ZOO" → Zoo
  - "HERITAGE" → Heritage
  - "WATERFALL" → Waterfall
  - "CASTLE" → Castle
  - "PARK" → Park
  - その他多数のキーワードマッピング

### FR-004-03: 観光/経費の振り分け

- SightseeigTypeの一部（Insurance, Ticket, Accident, Shopping, Medical, Washing, Tax, Exchange, Cashing, Haircut, Tips, PartTimeJob, Toilet）は「その他経費」カテゴリとして扱う
- 観光リストから自動的にOtherListへ振り分ける

### FR-004-04: 観光記録の一覧表示

- 国カラムには国コードではなく**国名**を表示する（FR-002-03 参照）

### FR-004-05: 観光タイプ別集計

- 観光タイプごとの件数・合計費用・最大・最小・平均費用を集計

## テスト観点

- キーワード→タイプ自動分類の全パターン網羅
- 大文字小文字の扱い（コンテキスト先頭文字の大文字変換）
- 観光/経費の振り分けロジック
- タイプ別集計の正確性
- コンテキスト文字列のサニタイズ
