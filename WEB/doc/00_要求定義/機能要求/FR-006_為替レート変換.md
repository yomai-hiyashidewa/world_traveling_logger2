# FR-006: 為替レート変換

## 概要

複数通貨での経費を主要3通貨（JPY, EUR, USD）に自動変換する。

## 要求一覧

### FR-006-01: 為替レートデータの読み込み

- 為替レートCSVを読み込み、通貨・レート・日付のセットとして保持する
- CurrencyType列挙値で通貨を識別

### FR-006-02: レート取得

- 指定通貨・指定日付に対応するレートを返す
- 日付範囲の平均レートを算出する機能

### FR-006-03: 全レコードの通貨変換

- 各記録（宿泊・交通・観光・その他）の料金を、記録日の為替レートに基づいてJPY・EUR・USDに変換する
- 変換後の金額を各レコードに付与

### FR-006-04: 表示通貨の切り替え

- ユーザーは表示通貨（JPY / EUR / USD）を切り替えて金額を閲覧できる

### FR-006-05: 通貨フォーマット

- JPY: 円マーク、小数なし
- EUR: ユーロマーク、小数2桁
- USD: ドルマーク、小数2桁

## テスト観点

- レートCSVの正常読み込みとパース
- 特定日付・通貨のレート取得
- 日付範囲での平均レート算出
- JPY/EUR/USD各変換計算の精度
- 通貨フォーマット文字列の出力形式
- レートデータが存在しない通貨・日付のハンドリング
