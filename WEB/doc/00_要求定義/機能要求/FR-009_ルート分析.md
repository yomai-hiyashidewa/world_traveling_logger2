# FR-009: ルート分析

## 概要

旅行ルート（入国・出国・国内地域間移動）を分析・表示する。
元コードの RouteView は3パネル構成で、入国一覧・国内移動経路・出国一覧を表示する。

## 要求一覧

### FR-009-01: 入国・出国分析

- 交通記録から各国への入国（Arrival）・出国（Departure）を特定する
- 入国: 到着国が対象国で出発国が異なる交通記録（NO_ENTRY除外）
- 出国: 出発国が対象国で到着国が異なる交通記録
- 元コード: TransportationList.GetArrivals(), GetDepartures()

### FR-009-02: 入国元・出国先一覧表示（左右パネル）

- カントリーモード時に、選択国への入国元の国一覧（左パネル）と出国先の国一覧（右パネル）を表示する
- 各国に国旗アイコンを表示する（FR-011-03）
- 国を選択すると、その国境越え移動の詳細（地域、交通手段、距離、時間）を表示する
- **初期表示時、最初の入国元/出国先を自動選択し、詳細を表示する**（元コード: `current_ = transportations_.FirstOrDefault()`）
- 詳細の表示内容:
  - 地域（入国: 出発地域、出国: 到着地域）
  - 交通手段（TransportationType）
  - 距離（"#,0km" 形式、例: "1,234km"）
  - 時間（"Xd Yh Zm" / "Xh Ym" / "Xmin" 形式、例: "1d 2h 30m"）
  - 日をまたぐ移動（startDate ≠ endDate）には "zzz" マーカーを表示
- 元コード: RouteCountryViewModel, RouteCountryViewPanel.xaml, MovingModel（距離・時間フォーマット）
- **ワールドモード時（国未選択）は非表示**

### FR-009-03: 国内移動経路表示（中央パネル）

- カントリーモード時に、選択国内での全移動経路を時系列で表示する
- 表示対象: 選択国に関わる全交通記録（startCountry または endCountry が選択国のもの）
  - 元コード: TransportationList.GetRoute(countryType) → `calcList_.Where(m => m.StartCountry == type || m.EndCountry == type)`
- 各レコードの表示内容:
  - **国内移動**（SameCountry = true）: 出発地域・出発場所 → 交通手段（距離・時間） → 到着場所・到着地域
  - **入国**（到着国 = 選択国、出発国 ≠ 選択国）: 入国日・入国地域の表示
  - **出国**（出発国 = 選択国、到着国 ≠ 選択国）: 出国日・出国地域の表示
- 日をまたぐ移動（AnotherDate）にはマーカーを表示する
- 元コード: RouteRegionMiniViewModel.Routes, RouteRegionMiniViewPanel.xaml

### FR-009-04: 国滞在期間の表示

- 選択国の滞在開始日（最初の交通記録の到着日）と滞在終了日（最後の交通記録の日付）を表示する
- 国旗と国名を表示する
- 元コード: RouteRegionViewModel.StartDate, EndDate

### FR-009-05: ワールドモード時の国境越え一覧

- 国未選択（ワールドモード）時は、全ての国境越え移動を時系列で一覧表示する
- 各レコードに出発国・到着国の国旗と国名を表示する（FR-011-03）
- NO_ENTRY の国境越えは視覚的に区別する

### FR-009-06: ルート表示における国名表示

- 国境越え一覧、入国元・出国先一覧の全ての国表示において国コードではなく**国名**を表示する（FR-002-03 参照）

### FR-009-07: NO_ENTRY国の識別

- 経由のみで実質的に入国していない国（メモに"NO_ENTRY"を含む）を識別
- ルート表示上で区別する（バッジ等）
- 入国（Arrival）リストからは除外する

### FR-009-08: ルートタブのモード別レイアウト

- **ワールドモード**（国未選択）:
  - 全世界の国境越え一覧のみ表示（FR-009-05）
- **カントリーモード**（国選択時）:
  - 3パネル構成:
    - 左パネル: 入国元一覧（FR-009-02）
    - 中央パネル: 国内移動経路（FR-009-03）+ 滞在期間（FR-009-04）
    - 右パネル: 出国先一覧（FR-009-02）

## 元WPFコードとの対応

| 元コード | 機能 | WEB版対応 |
|---|---|---|
| RouteView.xaml | 3カラムレイアウト（arrivals/regions/departures） | FR-009-08 |
| RouteCountryViewModel | 入国元・出国先の国リスト+詳細（初期自動選択） | FR-009-02 |
| MovingModel | 距離・時間の表示フォーマット（#,0km / Xd Yh Zm） | FR-009-02 |
| RouteRegionViewModel | 国の滞在期間ヘッダー | FR-009-04 |
| RouteRegionMiniViewModel.Routes | 国内移動経路リスト | FR-009-03 |
| TransportationList.GetRoute() | 選択国に関わる全交通記録取得 | FR-009-03 |
| TransportationList.GetArrivals() | 入国記録抽出 | FR-009-01 |
| TransportationList.GetDepartures() | 出国記録抽出 | FR-009-01 |
| IsWithCrossBorder | 左右パネル表示/非表示 | FR-009-08 |

## テスト観点

- 国境越え交通記録からの入国/出国判定
- 同一国出発・到着（国内移動）が国内移動経路に正しく表示されること
- NO_ENTRYフラグの検出と入国リストからの除外
- 国内移動経路の時系列順
- 入国記録（到着国=選択国、SameCountry=false）の表示
- 出国記録（出発国=選択国、SameCountry=false）の表示
- 日またぎ移動（AnotherDate）のマーカー表示
- ワールドモード↔カントリーモードの切替でレイアウトが変わること
- 入国元・出国先一覧の国選択時に詳細が表示されること
- 入国元・出国先一覧の初期表示時、最初の国が自動選択され詳細（地域、交通手段、距離、時間）が表示されること
- 距離が "#,0km" 形式でフォーマットされること（例: 1234 → "1,234km"）
- 時間が "Xd Yh Zm" / "Xh Ym" / "Xmin" 形式でフォーマットされること
- 日をまたぐ移動（AnotherDate）に "zzz" マーカーが表示されること（左右パネル含む）
